# Burger bot Spec
ベースとなるロボットはTurtlebot3 burgerです。Turtlebot3 burgerにカメラとマーカーなどを追加で搭載しています。

ベースのTurtlebot3 burgerのスペックは下記のサイトを参照ください。

https://emanual.robotis.com/docs/en/platform/turtlebot3/features/
